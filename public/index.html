California Notary Tools

Free tools for California notaries. Data from CA SOS active notary database (136,000+ records, updated nightly) and CPS HR exam schedule.

Tools

🔍 Commission Verification

Search by name to verify someone's notary commission status, number, and expiration date. No contact info included — verification only.

Audience: Employers, title companies, anyone verifying a notary's credentials

Embed: iframe on calnotaryclass.com/verify-commission/

📋 Commission Checker

Look up a commission number to check expiration status and get course recommendations (6-hour vs 3-hour refresher).

Audience: Existing notaries checking their renewal status

Embed: iframe on calnotaryclass.com/commission-checker/

📊 County Stats

Live notary counts per county with top cities and expiring commissions. Data for county landing pages.

Audience: SEO (indexable HTML fragment embeds on county pages)

📝 Exam Schedule

Upcoming California notary exam dates, locations, and registration links from CPS HR Consulting.

Audience: People preparing to take the state exam

Embed: HTML fragment on calnotaryclass.com/exam-schedule/

API Endpoints

GET /api/search?q=smith&county=37&limit=20   → Verify commission by name
GET /api/commission?number=2538491              → Check commission status + recommendations
GET /api/stats                                  → All county stats (JSON)
GET /api/stats/san-diego                        → Specific county stats (JSON)
GET /api/exams                                  → Upcoming exam sessions
GET /api/exams?county=37                        → Exams filtered by county
GET /api/fragment/county/san-diego              → County stats as HTML (for WP embed)
